Key Features

This demo showcases enterprise-grade authentication and authorization patterns.

OAuth2 + OpenID Connect

Complete OAuth2 authorization code flow with Keycloak as the identity provider supporting JWT tokens and refresh tokens.

Role-Based Access Control

Fine-grained authorization with ADMIN, MANAGER, USER, and GUEST roles controlling access to different API endpoints and web pages.

Session Management

JWT-based stateless authentication with automatic token refresh and session expiration handling for both web and API access.

RESTful APIs

Sample REST endpoints demonstrating different authorization levels with comprehensive API testing and documentation.

Docker Compose

Production-ready Docker setup with Keycloak, PostgreSQL, and automated realm configuration for easy deployment.

Enterprise Security

Production-ready security patterns with proper token validation, CSRF protection, and secure session management.
